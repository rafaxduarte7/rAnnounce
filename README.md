# 📢 rAnnounce - Plugin de Anúncios para Jogadores

Um plugin simples e personalizável de **anúncios** para jogadores em servidores Minecraft. Ideal para servidores que desejam permitir que jogadores anunciem mensagens no chat de forma controlada, com suporte a **permissões**, **cooldowns** e **configurações personalizadas**.

## ✨ Funcionalidades

- ✅ Comando de anúncio para jogadores
- 🔒 Sistema de permissões
- ⏳ Cooldown configurável entre anúncios
- ⚙️ Configuração de prefixos, cores e formatos
- 📁 Arquivo de configuração fácil de editar

## 📦 Instalação

1. Baixe a versão mais recente do plugin [aqui]([#](https://github.com/rafaxduarte7/rAnnounce/releases/latest)).
2. Coloque o arquivo `.jar` na pasta `plugins/` do seu servidor.
3. Certifique-se que o seu servidor possui java 8 ou suporte para o mesmo.
4. Reinicie ou recarregue o servidor.
